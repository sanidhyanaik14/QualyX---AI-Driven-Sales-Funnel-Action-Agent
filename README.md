# 🤖 AI Lead Classification Platform

A full-stack AI-powered platform that classifies leads into **Hot 🔥, Warm 🌤️, and Cold ❄️** using intelligent scoring powered by Large Language Models (LLMs).

---

## 🌟 Features

- 🔍 **AI-Based Lead Scoring**
  - Automatically classifies leads into Hot, Warm, and Cold categories

- 🔐 **Secure Authentication**
  - User authentication and data handling using Supabase

- 📊 **Personalized Dashboard**
  - Each user gets their own interactive dashboard

- ⚡ **Smart Prioritization**
  - Helps businesses focus on high-value leads first

- 🌐 **Full Stack Architecture**
  - Frontend built with Next.js  
  - Backend powered by Python

---

## 🛠️ Tech Stack

- **Frontend:** Next.js, React  
- **Backend:** Python  
- **Database & Auth:** Supabase  
- **AI:** LLM-based scoring
